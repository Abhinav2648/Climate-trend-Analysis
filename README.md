# 🌍 Climate Trend Analysis

## 📌 Project Overview

Climate Trend Analysis is a web-based data analytics project that examines historical climate datasets to identify temperature trends, regional patterns, and environmental changes over time.

This project helps users understand global climate behavior using interactive analysis and visual insights.

---

## 🚀 Features

* 📈 Analyze historical climate data
* 🌡️ Identify rising/falling temperature trends
* 📊 Data visualization using charts and graphs
* 🗺️ Region-wise climate insights
* 🌐 Web interface using Flask

---

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Abhinav2648/Climate-trend-Analysis.git
```

2. Move into folder

```bash
cd Climate-trend-Analysis
```

3. Install dependencies

```bash
pip install pandas flask matplotlib numpy
```

4. Run the project

```bash
python app.py
```
